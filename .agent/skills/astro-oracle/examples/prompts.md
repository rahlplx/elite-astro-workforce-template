# Astro Oracle: Example Prompts

This file contains ready-to-use prompts that leverage the Astro Oracle agent.

## Component Creation

### Basic Component

```text
Astro Oracle: Create a responsive Card component with TypeScript props for title, description, and an optional image.
```

### Server Island

```text
Astro Oracle: Create a Server Island component for displaying real-time user data with a loading fallback.
```

## Content Collections

Astro Oracle: Show me the correct way to define a content collection schema for a blog with tags and categories in Astro 5.

Astro Oracle: How do I filter and sort a content collection by date in Astro?

## Pro Tips

1. **Be Specific**: Include version numbers and error messages for better results.
2. **Use Keywords**: The agent indexes keywords like "content layer", "actions api", "server islands".
3. **Chain Requests**: Ask follow-up questions to drill down into specific details.

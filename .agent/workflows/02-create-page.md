## What This Does
Creates a new Astro page with:
- Proper file structure
- SEO meta tags
- Layout integration
- Responsive design
- Accessibility built-in

## How to Use

### Quick Start
```
Create a new About page
```

### With Details
```
Create a Contact page with a form, map, and business hours section
```

### With Specific Requirements
```
Create a Services page with 3 columns of service cards, each with icon, title, and description
```

## What You'll Get

1. **New .astro file** in `src/pages/`
2. **SEO meta tags** configured
3. **Layout wrapper** applied
4. **Responsive design** using Tailwind/DaisyUI
5. **Accessible markup** with proper headings

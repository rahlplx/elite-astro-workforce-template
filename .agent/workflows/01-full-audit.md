## What This Does
Performs a comprehensive audit of your entire Astro project, checking:
- Project structure and configuration
- Code quality and anti-patterns
- UI/UX accessibility issues
- SEO meta tags and structure
- Performance indicators
- Security concerns

## How to Use

### Quick Start
Just say:
```
Audit my project
```

### Detailed Audit
```
Run a full audit of my Astro project and give me a report with all issues
```

### Specific Audits
```
Audit my project for accessibility issues only
```
```
Check my SEO configuration
```
```
Scan for performance problems
```
